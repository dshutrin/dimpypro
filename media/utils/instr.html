1. Ваш проект должен соответствовать структуре:

  • < Директория проекта >
    • < Основное приложение >
      • settings.py
      • ...
      • wsgi.py
    • Приложение 1
    • Приложение 2
    • Приложение ...
    • Приложение X
    • static/
      • js
      • css
      • admin
    • media/
      • ...

2. Загрузка проекта

После запуска файла - вас поросят ввести ссылку на github репозиторий проекта. Репозиторий должен быть открытым для того, чтобы его можно было скачать. (Либо самостоятельно добавьте ssh ключи перед тем запускать утилиту)

3. Настройка

Далее у вас попросят ввести домен, на котором будет рабоотать сайт. Нужно ввести действующий домен, если его нет - ip адрес вашего сервера, на котором будет запущен проект.

4. Миграции

Миграции выполнятся автоматически, но вот создать суперпользователя вам придётся вручную.
